## テレビ電話
ディレクトリ[phone](./phone)には最終的に完成したテレビ電話のソースコードが入っています．
```
cd phone
make
```
でコンパイルできます．実行は，`./talk -l`でサーバとして起動，`./talk <サーバのIPアドレス>`でクライアントとしてサーバに接続できます．

実行する際にはWebカメラが接続・認識されている必要があります．
